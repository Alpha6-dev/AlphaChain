# AlphaChain Token (ACT)

Utility token for platform transaction fees on the AlphaChain network, built using the Hyperledger Fabric Token SDK.

## Token Details

- **Name**: AlphaChain Token
- **Symbol**: ACT
- **Decimals**: 2
- **Use**: Transaction fees, payment settlements, staking

## Functions

- `Initialize` — Set token name, symbol, decimals
- `Mint` — Mint new ACT tokens (admin only)
- `Transfer` — Transfer tokens between accounts
- `BalanceOf` — Query account balance
- `TotalSupply` — Query total token supply
