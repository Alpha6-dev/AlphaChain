# AlphaChain Dashboard

React 18 + Tailwind CSS stakeholder dashboard for shipment tracking.

## Features

- Shipment list with filtering and search
- Shipment detail view with status timeline
- Role-based views (Supplier / Transporter / Buyer / Customs / Admin)
- Real-time status updates via polling

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
